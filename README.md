# Meshtastic Game Hub (Offline Edition)

Автономна ігрова платформа для Meshtastic-мереж, яка дозволяє грати в ігри (Морський бій, Шахи, Шашки) прямо в браузері, передаючи ходи через радіоефір без доступу до Інтернету.

---

## 1. Архітектура
Система складається з двох ключових частин:

* **bridge.py (Python-сервер):** виступає в ролі "моста". Він з'єднує ваш Meshtastic-модем із браузером через WebSocket та роздає ігрові файли по HTTP.
* **\*.html (Ігрові клієнти):** вся ігрова логіка (правила, розрахунки, візуалізація) працює локально у вашому браузері. Через радіо передаються лише легкі текстові команди.

## 2. Підготовка та встановлення
### Вимоги:
* **Python 3.10+** (рекомендовано 3.14).
* **Meshtastic-модем**, підключений по USB.

### Встановлення:
1. Створіть папку проєкту і помістіть туди всі файли (, ігрові  файли, папки , , ).
2. Встановіть необхідні бібліотеки:
   ```bash
   python3 -m pip install meshtastic websockets pypubsub
   ```

## 3. Як користуватися
### Запуск сервера:
1. Підключіть модем до пристрою через USB.
2. Запустіть міст: `python3 bridge.py`
3. Сервер запустить веб-інтерфейс за адресою: `http://127.0.0.1:8890`.

### Гра в браузері:
1. Перейдіть за адресою `http://127.0.0.1:8890`.
2. Натисніть кнопку **"Підключити"**. Після появи зеленого повідомлення модем готовий.
3. Введіть **короткий нік (Short Name)** суперника (якщо залишити поле порожнім, повідомлення піде в загальний канал Broadcast).
4. Натисніть **"Готовий до гри"**. Після того, як ваш опонент зробить те саме, дошка активується.

## 4. Troubleshooting
| Проблема | Причина | Рішення |
| :--- | :--- | :--- |
| `OSError: could not bind` | Порт 8890 зайнятий | Переконайтеся, що інший процес не використовує цей порт. |
| `Modem not found` | Модем не підключений | Перевірте USB-кабель. |
| Немає з'єднання | WebSocket відпав | Перезапустіть `bridge.py` та оновіть сторінку. |
| Фігури не рухаються | Суперник не отримав пакет | Перевірте правильність нікнейму або стан Mesh-мережі. |

## 5. Додавання нових ігор
1. Створіть файл (наприклад, `new_game.html`).
2. Використовуйте унікальний префікс (наприклад, `NEWGAME:`) для команд.
3. Додайте цей префікс у кортеж `text.startswith(("MB:", "CHESS:", "NEWGAME:"))` у файлі `bridge.py`.
