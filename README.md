# Meshtastic Game Hub (Offline Edition)

Ця система є автономною ігровою платформою для Meshtastic-мереж. Вона дозволяє грати в ігри (Морський бій, Шахи, Шашки) прямо в браузері, передаючи ходи через радіоефір без необхідності доступу до Інтернету.

---

## 1. Архітектура
Система складається з двох ключових частин:

* **bridge.py (Python-сервер):** виступає в ролі "моста". Він з'єднує ваш Meshtastic-модем із браузером через WebSocket та роздає ігрові файли по HTTP.
* **\*.html (Ігрові клієнти):** вся ігрова логіка працює локально у вашому браузері. Через радіо передаються лише короткі текстові команди.

## 2. Підготовка та встановлення
### Вимоги:
* Python 3.10+
* Meshtastic-модем, підключений по USB.

### Встановлення:
1. Створіть папку проєкту і помістіть туди всі файли.
2. Встановіть бібліотеки:
   ```bash
   python3 -m pip install meshtastic websockets pypubsub
   ```

## 3. Як користуватися
1. Запустіть міст: `python3 bridge.py`
2. Відкрийте `http://127.0.0.1:8890` у браузері.
3. Натисніть "Підключити", введіть нік суперника та грайте!
